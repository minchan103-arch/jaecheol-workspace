## 2. 프로필 세팅 기준

- **프로필 사진**: 로고 (PNG-01.png 기준)
- **소개글 공식**: `[무엇을 하는 사람] + [왜 달라] + [행동 유도]`
  - 예: "새벽 경매장에서 직접 당도 재고 골라옵니다 🍊 | 지금 제철 → 링크"
- **링크**: 스마트스토어 또는 카카오채널

## 3. 콘텐츠 구조 (영보스 스타일 적용)

```
① 후킹 (첫 3초/첫 줄)
   → 단호한 문장. "이유는 하나다" 형식
   
② 현상 진단
   → 독자가 공감할 상황 콕 짚기
   
③ 제철삼촌의 기준
   → 우리가 다른 이유 (경매장, 당도 선별, 직접 검수)
   
④ CTA
   → "지금 링크에서" or "DM으로" (과하지 않게)
```

**예시:**
> 참외 샀는데 맛없는 이유. 판매자가 당도 안 재고 올리기 때문이다. 저는 경매장에서 직접 당도 재고 선별합니다. 다음 주 참외 들어옵니다.
