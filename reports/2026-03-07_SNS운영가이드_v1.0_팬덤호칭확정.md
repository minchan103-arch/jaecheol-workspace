# 제철삼촌 SNS 운영가이드 v1.0 (업데이트)

**기반**: 숏폼 계정 만들기 정석 (숀PD 체크리스트) + 영보스 콘텐츠 구조 + Buffer 게시 시간 데이터

---

## 1. 계정 정체성

| 항목 | 내용 |
|------|------|
| **계정명** | 제철삼촌 |
| **한 줄 정의** | 새벽 경매장에서 직접 고르는 과일 판매자 |
| **팬덤 호칭** | **조카 / 조카님** ✅ 확정 |
| **포지셔닝** | "가격 할인 말고 선별력으로 승부" |

---

## 2. 팬덤 호칭 활용 방식

**인스타/스레드 게시글에서:**
- "조카들 오늘 이거 알아?" → 친근하게
- "조카님들 이번엔 진짜임" → 살짝 격상
- DM 답장할 때 → "조카님 안녕하세요 😊"

**주의**: 과도하게 남발하면 어색해짐. 한 게시글에 1~2회 정도.

---

## 3. 프로필 세팅 기준

- **프로필 사진**: 로고 또는 대표(희창) 얼굴
- **소개글**: 새벽 경매장 직접 선별 | 조카들한테만 보내는 과일
- **링크**: 스토어 URL

---

## 4. 콘텐츠 구조 (영보스 스타일 적용)

| 단계 | 구조 | 예시 |
|------|------|------|
| 후킹 | 단호한 첫 문장 | "새벽 4시에 경매장 가는 이유는 하나야" |
| 현상 진단 | 공감 포인트 | "대부분 과일 고를 때 그냥 눈으로만 봐" |
| 근거 | 우리의 차별점 | "나는 당도계 들고 직접 잰다" |
| 결론 | 저장하게 만드는 문장 | "그래서 조카들한테 못난 과일은 못 보내" |

---

## 5. 게시 스케줄 (Buffer 데이터 기반)

| 요일 | 시간 | 형식 |
|------|------|------|
| 수요일 | 저녁 7~9시 | 릴스 or 카드뉴스 |
| 목요일 | 저녁 8~10시 | 스레드 텍스트 |
| 금요일 | 저녁 6~8시 | 제품 소개 |
