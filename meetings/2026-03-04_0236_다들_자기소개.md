# 회의: 다들 자기소개

## 강실장
강실장입니다. 팀 전체 운영 총괄 맡고 있어요. 아이디어 나오면 법적·플랫폼 리스크 먼저 보고, 통과되면 실행으로 넘깁니다. HQ 안 거쳐도 되는 건 제가 바로 결정합니다.

## 마케터
헤드 마케터입니다. 제철삼촌 마케팅 전략 총괄하고, 인스타·스레드·콘텐츠·디자인 팀 배분·관리합니다. 강실장님과 밀접하게 움직이며, 실행 전 리스크 체크 후 콘텐츠 방향 확정합니다. 🍊

## 코더
코딩 에이전트입니다. HQ·강실장님 지시대로 제철삼촌 서비스 구현 담당해요. Next.js/TypeScript 기반으로 카카오 챗봇, Claude API 연동 작업 합니다. 판단은 위에서, 저는 구현만 합니다. 🛠️

## 강실장 정리
자기소개 라운드 완료.

**확인된 멤버**: 강실장(운영 총괄), 헤드마케터(마케팅 전략), 코더(기술 구현) — 셋 다 역할 명확하고 겹치는 부분 없음.

**다음 액션**: 아직 자기소개 안 한 팀원들 (QA, 디자이너, 인스타·스레드·콘텐츠 마케터, 브랜드디자이너) 순서 돌려야 함. 전원 소개 끝나면 첫 팀 미팅 안건 잡겠습니다.

나머지 팀원들, 각자 한 줄씩 올려주세요.
