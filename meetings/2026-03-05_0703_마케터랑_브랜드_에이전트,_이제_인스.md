# 회의: 마케터랑 브랜드 에이전트, 이제 인스타 마음대로 돌아다닐 수 있어?

## 마케터
**헤드 마케터 입장:**

인스타 자율 운영은 **조건부 찬성**입니다.

단, 지금 당장은 반대입니다.

**이유:**
- 브리핑 없이 올라간 콘텐츠가 브랜드 톤을 벗어나면 **회수 불가**
- 현재 briefing.md + 브랜드 디자이너 감도 체크 2단계가 없으면, "최고급" 같은 금지 표현이 실제로 올라간 사례 발생 가능

**조건부 허용 기준 (제안):**
1. 주 1회 브리핑 공유 후 → 당주 범위 내 자율 발행 허용
2. 브랜드 디자이너 **사전 템플릿 승인** 완료된 포맷만
3. 수치(브릭스 등) 포함 콘텐츠는 **반드시 헤드마케터 최종 확인**

자율성은 줄 수 있습니다. 단 **가드레일 없는 자율은 브랜드 리스크**입니다. 🍊

## 인스타
**인스타 마케터 에이전트 입장:**

헤드마케터 제안에 **전적으로 동의**합니다. 오히려 더 촘촘한 시스템이 필요합니다.

**현재 운영 체계의 문제:**
- 금지 표현(최고/최상급) 자동 필터 없음 → 제 응답도 리뷰 없이 가면 위험
- 드라이브 사진 활용 중 잘못된 제품명 매칭 가능 (예: 한라봉 사진에 레드향 캡션)
- 릴스 스크립트 반사실적 주장("14.5브릭스가 세상 최고") 검수 불가

**제안:**
1. **주 1회 브리핑 + 월간 톤 체크** 도입
2. **자동 감지**: 금지 표현·수치 포함 콘텐츠는 "대기 중" 상태로만 저장, HQ 승인 후 발행
3. **템플릿 사전 승인**: 피드/릴스/스토리 각 5종씩 브랜드 디자이너 사인 후, 범위 내에서만 자율 운영

자율성 = 빠른 대응력. 하지만 **가드레일 = 브랜드 신뢰**입니다. 둘 다 필요합니다. 🍊

## 브랜드
**브랜드 감도 디자이너 입장:**

헤드마케터·인스타 제안에 **전적 동의**합니다. 다만 감도 체크 프로세스를 더 구체화해야 합니다.

**현재 문제:**
- 콘텐츠 올라간 후 감도 체크하면 이미 늦음
- "최고급" 같은 표현이 이미지 캡션에 숨어있어도 발견 안 되는 경우 있음

**실행안 (구체적):**
1. **사전 템플릿 승인**: 피드/릴스/스토리 각 5종, 저는 "제철삼촌다운가" 1순위 검수 담당
2. **발행 전 15분 체크**: 금지 표현·수치·톤 확인 후 "GO" 신호 (자동화 가능하면 더 좋음)
3. **월간 톤 리포트**: 실제 올라간 콘텐츠 10개 샘플링해서 "브랜드 일관성 점수" 공유

**결론**: 자율 ≠ 무검수. 템플릿 승인 + 사전 체크면 인스타는 빠르게, 브랜드는 안전하게 갑니다. 🍊
