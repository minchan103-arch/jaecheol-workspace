# 회의: 3월 초 제철 과일 시즌 공백기(딸기 피크 종료, 한라봉·천혜향 마무리)에 맞춰 **다음 메인 시즌 품목(딸기 후속 or 봄 과일)**을 확정하고 마케팅 준비 일정을 세팅하는 것.

## 강실장
**강실장 발언**

헤드마케터가 제시한 두 트랙(참외 + 망고) 방향 동의한다. 다만 실행 전에 두 가지 체크.

첫째, **식품 과장 광고 리스크** — "국내산 제주 망고" 강조하려면 원산지 증명 가능한 농가 계약 먼저 확보해야 한다. 계약 없이 "제주산"으로 마케팅하면 식품위생법 위반.

둘째, **참외 론칭 타이밍** — 4월 중순 오픈이면 콘텐츠 제작→QA→배포까지 역산하면 지금 당장 작업 시작해야 한다. 3월 2주차에 인스타 초안이 올라와야 일정이 맞는다.

결론: 참외 메인, 망고 서브로 우선순위 확정하고, **코더에게 상품 페이지 세팅 일정 확인, 인스타 마케터에게 참외 콘텐츠 초안 3월 10일까지 요청** 두 개 바로 넣겠다.

HQ, 품목 확정 최종 승인 한 번만 해주시면 된다.

## 마케터
# 헤드마케터 발언

강실장 체크포인트 수용한다. 타임라인 역산해서 구체화한다.

**참외(메인):** 4월 중순 출시 기준 → 3월 10일 인스타 초안 → 3월 17일 브랜드 디자이너 감도 검수 → 3월 24일 스레드 콘텐츠 완성 → 4월 1일 전체 QA 완료. 이 일정 맞추려면 **이번 주 내로 인스타 마케터에게 브리핑 인박스 전달**한다.

**망고(서브):** 제주산 계약 농가 확보 전까지 마케팅 보류. 대신 이 기간에 "망고 선별 기준 / 브릭스 수치" 콘텐츠 예열만 준비 → 농가 계약 완료 즉시 론칭 가능하도록 스탠바이.

**핵심 메시지(참외):** "경매장에서 직접 골랐다, 단 참외만"

**HQ 확인 요청:** 참외·망고 품목 최종 승인 + 제주 망고 농가 계약 진행 주체(강실장 or HQ 직접) 확인 필요.

## 코더
참외 상품 페이지 기준으로 역산하면, **4월 1일 QA 완료 맞추려면 3월 24일까지 페이지 배포 완료**돼야 한다.

그러면 제 작업 마감은 **3월 21일** — 상품 페이지 구현 + 내부 검토 시간 3일 확보.

지금 당장 필요한 건 **상품 정보 확정** (품명, 가격, 중량 옵션, 대표 이미지 유무). 이게 3월 10일 인스타 초안과 같이 나오면 그때 바로 작업 시작 가능.

망고는 농가 계약 완료 시점 알려주시면 역산해서 일정 잡겠다. 지금은 대기.

**HQ께 확인할 게 하나 있어요** — 참외 상품 페이지, 기존 상품 페이지 템플릿 그대로 쓰면 되나요, 아니면 새 레이아웃 필요한가요? 이게 작업량이 꽤 달라져서요.

## 강실장 정리
**결론**: 참외(메인) 4월 중순 론칭, 망고(서브) 농가 계약 후 론칭으로 품목 우선순위 확정 — 지금 당장 참외 콘텐츠·페이지 제작 착수.

---

**오늘 바로 할 일**:

- **인스타 마케터**: 참외 콘텐츠 초안 3월 10일까지 제출. 핵심 메시지 "경매장에서 직접 골랐다, 단 참외만". 헤드마케터 브리핑 인박스 이번 주 내 수령 후 시작.
- **코더**: 참외 상품 페이지 작업 시작 준비. 3월 10일 인스타 초안과 함께 상품 정보(품명·가격·중량 옵션·대표 이미지) 확정되면 즉시 착수, 3월 21일 완료 목표.
- **브랜드 디자이너**: 3월 17일 참외 인스타 콘텐츠 감도 검수 일정 블로킹.
- **헤드마케터**: 망고 예열 콘텐츠(선별 기준/브릭스 수치) 스탠바이 준비. 농가 계약 완료 전까지 "제주산" 표현 사용 금지 — 계약 확보 후 즉시 론칭 가능 상태로 대기.

---

**HQ 확인 필요**:

1. **참외·망고 품목 최종 승인** — 팀 전체 합의 완료, HQ 확인만 남음.
2. **제주 망고 농가 계약 진행 주체** — 강실장이 진행할지 HQ 직접 할지 결정 필요. *(식품위생법 원산지 증명 이슈 있어서 계약 주체 명확히 해야 함)*
3. **참외 상품 페이지 레이아웃** — 기존 템플릿 재사용 vs 신규 레이아웃. 작업량 차이 큼. 코더 3월 21일 마감 지키려면 이번 주 내 결정 필요.
