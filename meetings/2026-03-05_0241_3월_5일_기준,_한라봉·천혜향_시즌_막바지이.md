# 회의: 3월 5일 기준, 한라봉·천혜향 시즌 막바지이고 레드향은 피크인데 — 아직 레드향 콘텐츠 파이프라인이 세팅 안 됐으니까 오늘 안건은 이거다:

**"레드향 시즌 피크 대응 콘텐츠 파이프라인 —

## 강실장
**강실장:**

리스크 체크 먼저. 레드향 관련해서 "면역력", "다이어트" 같은 기능성 표현 쓰면 식품위생법 위반이다. "달다", "즙이 많다", "껍질이 얇다" — 맛과 식감 위주로 가야 해. AI 생성 이미지 쓸 거면 Meta 정책상 표시 의무 있으니까 콘텐츠마다 확인할 것.

실행안은 이렇게 본다. 오늘 마케팅 팀이 레드향 콘텐츠 초안 3종 뽑고, Tech 팀이 카카오채널 발송 세팅 확인. 마감은 내일 오전까지. 피크 시즌 2주 남았으면 지금 안 움직이면 늦는다.

헤드마케터, 인스타, 스레드 — 각자 포맷 하나씩 맡아줘. 중복 없이.

## 마케터
**[헤드마케터 발언]**

강실장 방향 맞다. 포맷 분배 먼저 확정한다.

- **인스타**: 선별 장면 릴스 — "100개 중 23개 탈락" 실제 수치 포함, 비파괴 당도 측정 장면 삽입
- **스레드**: "레드향이 한라봉보다 얇은 이유" 정보성 텍스트 — 껍질 두께·즙 비율 수치 근거
- **콘텐츠 제작**: 시즌 마감 긴급감 뉴스레터 — "레드향 피크, 2주 남았습니다"

핵심 메시지 한 줄: **"올해 마지막 레드향, 삼촌이 직접 고른 것만 갑니다."**

브릭스 수치 확보 여부 지금 바로 확인 필요 — 수치 없으면 신뢰 메시지 절반 날아간다. 강실장이 언급한 기능성 표현은 전 채널 금지어로 briefing에 박겠다. 내일 오전 마감 맞춘다.

**HQ 확인 필요**: 현재 레드향 재고 수량 — 품절 임박 메시지 쓸 수 있는지 판단 필요.

## 코더
지금 파이프라인 세팅 안 된 게 기술 쪽 문제인지 확인하면서 말한다.

레드향 콘텐츠 발송 관련해서 카카오채널 세팅 상태 지금 점검했다. 현재 카카오채널 API 연동은 되어 있는데, **콘텐츠 템플릿이 등록 안 된 상태**다. 발송하려면 카카오 비즈니스 채널에서 메시지 템플릿 심사 통과해야 하는데, 심사 평균 1~3 영업일 걸린다. 오늘 바로 초안 올려도 내일 오전 발송은 불가능한 구조다.

**실행안**: 카카오 템플릿 심사는 오늘 중 제출하고, 심사 대기 기간 동안 인스타·스레드 먼저 집중한다. 뉴스레터는 카카오 대신 현재 바로 발송 가능한 채널 있으면 HQ가 확인해줘야 한다.

**HQ 확인 필요**: 카카오 외에 즉시 발송 가능한 채널(문자, 이메일 등) 있는지 — 있으면 오늘 안에 세팅한다.

## 강실장 정리
**결론**: 레드향 피크 2주, 오늘 콘텐츠 초안 3종 착수 + 카카오 템플릿 심사 제출로 채널별 병렬 대응한다.
