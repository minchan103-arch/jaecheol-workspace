## 강실장
**좋아. 리스크 체크 먼저.**

"조카" 호칭이랑 삼촌 말투 — 법적 문제 없음. 플랫폼 정책도 통과.

단, 판매 과일 한정 타겟팅은 **재고 연동이 핵심 변수**야. 콘텐츠 올렸는데 품절이면 브랜드 신뢰 깎임.

→ 코더한테 확인 필요: 현재 판매 상품 상태 실시간 반영 가능한지.

말투 일관성은 **스레드 톤 기준으로 가이드 문서 하나 만들어야** 나중에 흔들리지 않음. 콘텐츠팀한테 첫 작업으로 잡을게.

## 마케터
**헤드 마케터 의견**

강실장 리스크 체크 동의. 추가로 —

"조카" 호칭은 **첫 콘텐츠에서 선제적으로 브랜드 선언** 해야 함. "우리 조카들한테 파는 거니까" 한 줄이 뻔뻔함을 무기로 만듦. 어색하게 스며들면 오히려 어색해짐.

판매 과일 타겟팅은 **품절 24시간 전 콘텐츠 스케줄 컷 룰** 만들면 재고 연동 전에도 운영 가능. 코더 개발 대기 중에도 수동으로 막을 수 있음.

**스레드 톤 가이드 문서** → 콘텐츠팀 첫 작업 동의. 여기에 "삼촌이 절대 안 쓰는 말" 리스트(최고급, 프리미엄 등) 함께 포함 요청.

**이번 주 핵심 메시지 후보:** *"삼촌이 직접 골랐으니까, 조카는 그냥 먹으면 돼."*

HQ 확인 필요: 첫 선언 콘텐츠 채널 우선순위 (스레드 선발 vs 인스타 동시 오픈).

## 인스타
**[인스타 담당 의견]**

강실장·헤드 마케터 의견 전부 동의합니다. 추가로 인스타 관점에서:

**1. 첫 선언은 인스타 피드 우선 (근거)**
- 스레드는 텍스트 중심 → "조카" 호칭 설명에 최적
- 인스타 피드는 **이미지 임팩트 + 캡션 3줄** → "뻔뻔함"이 비주얼로 먼저 전달됨
- 릴스 1개(15초, "조카 과일 고르는 삼촌" 콘셉트) + 피드 1개 동시 오픈 추천

**2. 판매 과일 타겟팅 운영안**
- 현재 판매 상품 3~5개만 로테이션 콘텐츠 대상
- 릴스/피드 게시 24시간 후 재고 체크 → 품절 시 스토리 "sold out 감사합니다" 1장으로 대체
- 이미지 드라이브는 상품별로 미리 촬영 (현재 분량으로 충분)

**3. 톤 가이드 문서**
- "절대 금지": 최고급·세상에서 제일·할인강조 + **"삼촌이 추천한다"는 거창한 표현**
- "필수 포함": 일상 톤 + 조카에게 말 거는 느낌 (예: "이거 진짜 ~~더라")

**첫 콘텐츠 실행 시기:** HQ 확인

## 스레드
**스레드 마케터 의견**

강실장·헤드마케터·인스타팀 의견 전부 동의. 스레드 관점 추가:

**1. 스레드 선발 실행 근거 (재검토)**
인스타가 "첫 선언은 인스타 우선"이라고 했는데, 스레드 특성상 **"조카" 호칭 선언은 스레드가 더 강함**. 이유:
- 스레드는 댓글이 알고리즘 = 즉시 피드백 가능 (24시간 내 반응으로 인스타 타이밍 조정 가능)
- 텍스트만으로 바이럴 = "뻔뻔함"이 말투로 직접 전달됨 (이미지 임팩트 필요 없음)
- 스레드 먼저 → 반응 좋으면 인스타 피드/릴스로 '확산' 구조 추천

**2. 판매 과일 타겟팅 스레드 운영안**
- 첫 콘텐츠: "조카 선언" (상품 미언급, 톤만)
- 이후 주 3~4회: 현재 판매 과일 1종씩 (연속 스레드 2/3형으로 스토리화)
- 품절 24시간 전 스케줄 컷 + 재고 없으면 '조카들이 다 가져갔다' 식 자조 톤으로 처리
