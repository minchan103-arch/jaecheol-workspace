# 회의: 3월 초 제철 과일인 **레드향 시즌 마감 전**, 잔여 재고 소진을 위한 라스트콜 콘텐츠 전략을 이번 주 안에 확정하고 실행 준비까지 완료하는 것.

## 강실장
**리스크 체크 먼저.**

레드향 콘텐츠에서 걸리는 게 두 가지야.

첫째, "마지막 기회" 류 문구는 허위 긴박감 조성으로 공정위 회색지대. "이번 주 마감 예정" 같이 실제 재고 상황 기반 표현으로 써야 해.

둘째, 식품 관련 효능 표현 — "달다", "맛있다"는 괜찮은데 "피로 회복", "비타민 풍부해서 건강에 좋다" 수준 넘어가면 식품위생법 걸려. 선 명확히 그어두자.

이 두 가지 통과하면 이번 주 실행까지 매니저 선에서 승인 가능해. 마케팅 팀, 안 이상으로 초안 잡아와. 목요일까지.

## 마케터
**[헤드 마케터 — 회의 발언]**

강실장 리스크 기준 수용. 두 가지 레드라인 팀 전체 공유하겠습니다.

이번 레드향 라스트콜 핵심 메시지는 **"재고 소진 아닌, 시즌 마감"** 프레이밍입니다. "이번 주가 마지막 경매 입고분"이라는 실제 사실 기반으로 긴박감을 만들 것.

**채널별 배분:**
- 인스타: 선별 탈락컷 + "이 중에서 남은 것" 시각화 릴스
- 스레드: "레드향은 왜 3월이 끝인가" 정보성 텍스트 (수확 시기/당도 하락 근거 포함)
- 수치 필수 삽입 — 브릭스 13.5 이상 기준, 탈락률 xx%

**목요일 마감 역산:**
- 화요일 오전: 각 채널 초안 완성
- 수요일: 브랜드 디자이너 감도 체크
- 목요일: 강실장 최종 승인

효능 표현은 "새콤달콤한 맛" 선에서 완전 차단. 건강 언급 제로로 가겠습니다. 초안 목요일 가져오겠습니다.

## 코더
**회의 발언 — 코더**

마케터 안 보면서 콘텐츠 실행 지원 관점에서 하나만.

"이번 주 마감 예정" 같은 실시간 재고 연동 표현 쓰려면, 현재 구조상 Google Sheets에서 재고 수량 끌어와서 프론트에 노출하는 게 가장 빠릅니다. 하루 한 번 수동 업데이트라도 Sheets에 수량 컬럼 하나 추가하면 카카오 챗봇이나 웹에서 "현재 xx박스 남음" 텍스트로 보여줄 수 있어요.

목요일 초안 맞추려면 화요일까지 Sheets 구조 확정해 주시면 수요일 내로 연동 붙일 수 있습니다. 재고 어디서 관리하시는지 확인 필요합니다.
