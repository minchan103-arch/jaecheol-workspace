# 회의: 3월 제철 과일(레드향·천혜향·한라봉) 시즌 마감이 임박한 시점에서, 잔여 시즌 기간 동안 판매 전환율을 최대화하기 위한 **마감 임박 콘텐츠 전략** 수립.

## 강실장
**리스크 체크 먼저.**

식품 "마감 임박" 표현은 괜찮은데, "오늘만 이 가격" 같은 허위 긴급성은 공정위 제재 대상이야. 실제 재고 기반으로 수량 명시해야 해.

실행 방향은 단순하게: **재고 수량 + D-카운트다운 조합**. "레드향 잔여 47박스" 이런 식으로. 숫자가 있어야 신뢰가 살고 전환이 된다.

콘텐츠 파이프라인은 인스타 → 스레드 순으로 하루 1건씩. AI 이미지 쓰면 표시 붙여야 하니 실제 제품 사진 우선 쓰는 게 낫다. 불필요한 리스크는 없애고 가자.

## 마케터
**[헤드 마케터 발언]**

강실장 방향 동의. 여기에 채널 역할 분리 추가합니다.

**인스타**: 실제 선별 현장 사진 + "레드향 잔여 47박스, 브릭스 13.2 이상만 남았습니다" — 숫자로 희소성 구현.

**스레드**: "3월이 끝나면 이 맛은 내년까지 없습니다" 시즌 서사로 감성 전환 유도.

두 채널이 **신뢰(수치) + 아쉬움(계절감)** 각각 담당, 메시지 충돌 없이 시너지.

핵심 메시지 한 줄: *"숫자가 줄고 있습니다. 계절도 함께."*

재고 수치는 실시간 연동 필요 — CS팀 또는 HQ에서 일 1회 업데이트 주시면 콘텐츠 즉시 반영하겠습니다.

## 코더
재고 수치 실시간 연동 얘기 나왔으니까 기술 쪽에서 한마디 하겠습니다.

"일 1회 수동 업데이트"로 시작하되, Google Sheets에 재고 컬럼 하나 추가하면 됩니다. 현재 lib/sheets.ts 이미 연결돼 있으니 추가 인프라 없이 가능해요.

인스타/스레드 콘텐츠용 API 엔드포인트 하나 만들면 — `/api/stock` — 마케터가 Sheets에 숫자 입력하면 챗봇이나 콘텐츠 자동 반영까지 연결 가능합니다. 개발 공수는 반나절 이내.

HQ가 실시간 자동화까지 원하는지, 수동으로 갈지만 결정해 주시면 바로 구현합니다.

## 강실장 정리
## 일일 전략 회의 결과 — 제철 과일 시즌 마감 임박 콘텐츠 전략

---

**결론**: 실제 재고 수량 + D-카운트다운 조합으로 인스타(신뢰/수치)·스레드(감성/계절감) 채널 역할 분리해 하루 1건씩 즉시 집행, 재고 수치는 수동 일 1회 업데이트로 시작하고 자동화 여부는 HQ 판단 후 코더가 반나절 이내 구현.

---

**오늘 바로 할 일**:
- (헤드마케터: 레드향·천혜향·한라봉 현재 잔여 재고 수량 오늘 중 확보 — 수치 없으면 콘텐츠 못 나감. 이후 일 1회 Google Sheets 업데이트 루틴 담당)
- (인스타 마케터: 잔여 재고 수량 + 브릭스 수치 받는 즉시 피드 초안 작성 — "레드향 잔여 OO박스, 브릭스 13.2 이상만 남았습니다" 형식. "오늘만 이 가격" 등 허위 긴급성 표현 절대 금지. 실제 선별 사진 사용, AI 이미지 쓰면 Meta 표시 의무 발생하니 피할 것)
- (스레드 마케터: 인스타와 메시지 충돌 없이 감성·계절감 담당 — "3월이 끝나면 이 맛은 내년까지 없습니다" 방향으로 초안 작성. 외부 링크는 초반 게시물에 넣지 말 것)
- (코더: Google Sheets 재고 컬럼 추가 + `/api/stock` 엔드포인트 구현 준비 — HQ 자동화 여부 결정 나오면 바로 착수할 수 있도록 설계안만 오늘 중 정리해둘 것)

---

**HQ 확인 필요**: 재고 수치 자동화 여부 — 수동(현행 유지) vs 실시간 자동화(`/api/stock` 연동) 중 방향 결정 필요. 자동화 선택 시 코더가 반나절 이내 구현 가능.
