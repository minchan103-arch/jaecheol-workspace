## 2026-03-05 10:59 — 팀 회의
**강실장**: 팔로워 늘리기, 방향은 맞는데 방법이 중요해.

자동 팔로우/언팔, 댓글 도배 — 이거 지금 인스타 알고리즘이 바로 잡아. 쉐도우밴 맞으면 회복에 3~4주 걸린다. 우리 그 리스크 감당 못해.

현실적인 방법: **릴스 주 2회 + 저장 유도 콘텐츠**. 저장수 높으면 알고리즘이 밀어준다. 팔로워보다 도달률 먼저 올리는 게 순서야.

목표 수치 잡기 전에 — 지금 계정 팔로워 몇 명이고, 저장율·도달률 어떻게 돼? 그거 보고 얘기하자.
**마케터**: **헤드 마케터 의견**

강실장 방향 동의. 단, 팔로워 수 자체를 KPI로 삼으면 안 된다.

**이번 달 실행안:**
- 릴스 주 2회 → "선별 과정" 시리즈 (경매장 새벽 + 탈락 과일 이야기)
- 저장 유도 문구 통일: "이 기준 저장해두세요 🍊"
- 스레드 연동 — 릴스 못 본 층에 텍스트로 동일 서사 재배포

**수치 목표 (현황 파악 후 확정):**
- 저장율 3%→5% 이상
- 도달률 주 +15% 이상
- 팔로워는 결과로 따라오는 것

**지금 당장 필요한 것:** 현재 계정 인사이트 수치. 강실장 말대로 그게 먼저입니다. 데이터 공유되면 채널별 배분 즉시 잡겠습니다.

지금 인스타 인사이트 수치 공유받는 것, 구글 시트에 자동으로 쌓이게 만들 수 있어요. 매주 수동으로 캡처하거나 옮기는 거 없애고, 저장수·도달률·팔로워 증감 한눈에 보이는 대시보드.

릴스 주 2회 올리면 데이터가 쌓이는데, 그 데이터로 "어떤 콘텐츠가 저장율 높았나" 바로 판단할 수 있어야 방향 수정도 빠르게 됩니다.
