# 제철삼촌 3월 천혜향 릴스 콘티

**제목:** "3월에도 제철 과일이 있다는 거 아세요?"  
**길이:** 25초

---

## [콘티]

**0~3초:** 
- 장면: 천혜향 한 개를 손에 들고 클로즈업 → 가로로 자르는 동작
- 자막: "🍊 3월도 제철이 있어요"
- 배경음: 트렌디한 경쾌한 사운드 시작

**3~15초:**
- 장면: 천혜향 단면의 주스가 흐르는 모습 (슬로우 모션)  
  → 숟가락으로 떠먹는 클로즈업
- 자막: "한라봉, 레드향 졌으면  
천혜향이 제철입니다  
(달콤함은 계속)"

**15~25초:**
- 장면: 가족이 함께 먹는 모습 또는 테이블에 여러 개 담긴 천혜향  
  → 제철삼촌 로고/계정 태그 애니메이션
- 자막: "봄 과일은 제철삼촌에서  
링크 클릭 👆"
- 배경음: 음악 자연스럽게 페이드아웃

---

## [자막 전체]

```
🍊 3월도 제철이 있어요

한라봉, 레드향 졌으면
천혜향이 제철입니다
(달콤함은 계속)

봄 과일은 제철삼촌에서
링크 클릭 👆
```

---

**포인트:**
- "3월인데 과일 제철이 끝났다고?" 오해 반박 → 천혜향 소개로 자연스럽게 전환
- 시각적 임팩트: 주스가 흐르는 장면으로 신선함 강조
- 구매 심리: "계속된다" = 질 좋은 제철 과일이 언제나 있다는 신뢰감
