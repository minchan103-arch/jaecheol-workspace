## 📌 스니콘 핵심 4가지 (제철삼촌 맞춤 적용)

### 1️⃣ **후킹 "표절" vs "응용" (★ 우리는 이걸 해야 함)**

**스니콘 말**: "남의 후킹, 그대로 표절하지 말고 나만의 스타일로 응용해라"

**제철삼촌 적용**:
- ❌ "당도를 재고 과일을 고르는데..."  (평범한 설명)
- ✅ "3000개 중에 **이거 빼달라고 해야 팔려**" (질문/공감형)
- ✅ "마트 과일이 다 짭인 이유? **당도 기준이 없어서**" (문제 지적)

**우리가 활용할 후킹 패턴**:
- 질문형: "한라봉이 정말 달까?" → 3초 안에 답변 시작
- 반전형: "우리 창고가 터졌어요. 왜냐하면..." → 재미 + 공감
- 데이터형: "3000박스 중 우리가 고른 건 300개" → 신뢰성

---

### 2️⃣ **대본 구조: "문제 → 공감 → 해결" (검증된 공식)**

**스니콘이 말한 구조**:
```
문제: 다들 빨래 냄새 왜 나는지 모르시죠?
공감: 사실 대부분 같은 실수합니다.
해결: 이 순서만 바꾸면 해결돼요.
```

**제철삼촌 릴스에 적용** (예시):
```
[0~3초] 문제/후킹: 
  "마트 토마토? 50% 확률로 신맛만 해요"

[3~10초] 공감:
  "우리도 처음엔 그랬어요. 경매장에서 
   하나하나 직접 만져보고..."

[10~20초] 해결:
  "당도로 고르니까? 아예 달라더라고요"

[20~30초] 클로징:
  "이제 매번 대저토마토 사용해요"
  (링크 유도 or 댓글 유도)
```

---

### 3️⃣ **포맷 선택: 릴스 > 캐러셀 > 스토리 (2026년 기준)**

**스니콘 인사이트**: "2026년은 캐러셀(카드뉴스)의 성적이 좋아지고 있다"

**제철삼촌 전략**:
- **릴스** (주 1~2회): 후킹+감정 + 판촉성 강함
  - 대저토마토 고르는 영상 / 당도 측정 모습 / 포장 비하인드
  
- **캐러셀** (주 1회): 정보성 + 신뢰 구축
  - "한라봉 vs 천혜향" 비교 / "당도 측정 방법" / "우리가 고르는 기준 5가지"
  
- **스토리** (매일): 팔로워 관계 유지
  - 오늘의 경매장 풍경 / 투표 스티커 (당도 맞추기) / 24시간 flash sale

---

### 4️⃣ **정보성 + 스토리텔링 조합 (찐팬 만들기)**

**스니콘 말**: "정보글만 올리면 팔로워만 모이고, 찐팬은 안 생긴다"

**제철삼촌 콘텐츠 믹스** (매주):
- **정보성 40%**: 당도 기준 / 선별 방법 / 보관 팁
- **스토리텔링 40%**: "왜 우리는 새벽 4시에 경매장 가나?" / "3000개 중 300개 고르는 이유"
- **라이프스타일 20%**: 포창과 서현의 일상 / 팀 뒷이야기 / 고객 반응 영상

## 🎬 **이번 주 적용 릴스 아이디어** (3개)

### 릴스 1️⃣: 대저토마토 당도 비교 (문제 → 공감 → 해결)
```
[0~3초] 후킹:
  "마트 토마토 vs 우리 토마토. 
   어디가 더 달까요?" (자막 + 영상)

[3~15초] 공감/비교:
  - 손으로 직접 당도계 접촉하는 영상
  - "마트 건 13도, 우리 건 18도"
  - "5도 차이가 얼마나 큰지 알아요?"

[15~25초] 결과:
  - 직접 먹고 반응 영상
  - "완전 달라요"

[25~30초] CTA:
  - "지금 쿠팡/스마트스토어 링크 👆"
```

### 릴스 2️⃣: 3000박스 창고 포화 (반전 + 긴급성)
```
[0~3초] 후킹:
  "어제 창고가 터졌습니다 ㅋㅋ" (자조적 톤)

[3~15초] 문제 전개:
  - 3000박스 적재 영상 (빠른 컷)
  - "8평 창고는 한계가 왔고..."

[15~25초] 현재 상황:
  - 포창 / 서현 멘붕 표정
  - "지금 팔아야만 합니다 ^^"

[25~30초] 기회 제시:
  - "대량주문 문의 DM 주세요"
```

### 릴스 3️⃣: 한라봉 당도 기준 설명 (정보성 + 신뢰)
```
[0~3초] 후킹:
  "한라봉이 정말 모두 달까요?" (질문형)

[3~20초] 설명:
  - 당도계 들고 경매장 영상
  - "우리 기준: 최소 11도 이상"
  - "시장 평균: 8~9도"

[20~28초] 신뢰포인트:
  - "만족 못하면 100% 환불"
  - 고객 리뷰 자막 1~2개

[28~30초] CTA:
  - "링크 👇 / 댓글로 질문해주세요"
```
