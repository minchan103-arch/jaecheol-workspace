# 제철삼촌 봄 딸기 릴스

**제목:** "딸기 제철, 지금이 가장 달 때"  
**길이:** 25초

---

## [콘티]

**0~3초:** 
- 장면: 아삭한 딸기를 깨물어 물리는 클로즈업 + 즙이 튀는 연출
- 자막: "딸기 언제 먹으면 제일 달까?"
- 배경음: 밝고 경쾌한 팝/트렌드 사운드

**3~15초:**
- 장면: 제철삼촌 선별장 딸기 다단계 접근샷 (전체→중간→클로즈)
  - 당도계 측정 장면 (숫자 보이게)
  - 딸기 꼭지 따는 손가락 동작
  - 반으로 자른 딸기 내부 선명도
- 자막: "3월 딸기가 당도 최고점 / 직접 선별한 이유 / 신맛은 빠지고 단맛만"

**15~25초:**
- 장면: 제철삼촌 딸기 팩 클로즈업 + 스택 쌓기 + 택배박스 담기
- 자막: "지금 주문하면 내일 도착 / 링크에서 확인"
- CTA: 화면 좌하단에 "링크 클릭" 또는 계정 태그

---

## [자막 전체]

```
딸기 언제 먹으면 제일 달까?

3월 딸기가 당도 최고점
직접 선별한 이유
신맛은 빠지고 단맛만

지금 주문하면 내일 도착
링크에서 확인
```

**배경음악 추천:** 
- 트렌드: Royalty Free 경쾌한 팝 (인스타 기본음악 라이브러리: "Spring Day", "Happy Vibes" 카테고리)

**참고:**
- 제철 감 유지: "가장 달 때"라는 시간성 강조 → 지금 구매 심리 자극
- 봄 콘셉트: 밝고 신선한 컬러, 자연스러운 햇빛 활용
- 클로즈업 → 중거리 → 상품 흐름으로 구매욕 유도
